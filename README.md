# Projeto de Dashboard/Relatório Power BI

Este repositório contém o projeto desenvolvido como parte do curso da DIO na área de Business Intelligence. O objetivo deste projeto foi criar um dashboard interativo e um relatório analítico utilizando o Power BI, com base em um conjunto de dados fornecido.

## Descrição do Projeto

O projeto consiste em um dashboard interativo que apresenta insights e visualizações a partir de um conjunto de dados específico. Através do Power BI, foram criadas várias visualizações e relatórios para facilitar a tomada de decisão, oferecendo uma análise clara e objetiva dos dados.

As principais funcionalidades do dashboard incluem:
- **Visualizações interativas**: Gráficos dinâmicos para explorar diferentes dimensões dos dados.
- **Análises de tendência**: Análises temporais e de variação de métricas ao longo do tempo.
- **Indicadores chave de desempenho (KPIs)**: Métricas importantes para a avaliação de desempenho do negócio.


## Objetivos do Projeto

- Aplicar os conhecimentos adquiridos no curso de Power BI.
- Desenvolver dashboards e relatórios eficientes para análise de dados.
- Utilizar boas práticas de visualização de dados.
- Criar relatórios com uma interface amigável e intuitiva.

## Ferramentas Utilizadas

- **Power BI Desktop**: Para criação de relatórios e dashboards.
- **Conjunto de Dados**: Sample de finanças disponibilizada pelo próprio Power BI.
